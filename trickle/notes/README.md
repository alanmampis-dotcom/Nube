# SecureCloud Project

Este proyecto es un panel de gestión en la nube con un sistema de autenticación seguro.

## Estructura del Proyecto

- `index.html`: Página de inicio de sesión con efecto de encriptación y 2FA.
- `dashboard.html`: Panel principal con gestión de archivos y gráficas.
- `app.js`: Lógica principal para la página de login.
- `dashboard-app.js`: Lógica principal para el dashboard.
- `components/`: Componentes reutilizables de React.
- `utils/`: Utilidades como validación de contraseñas.
